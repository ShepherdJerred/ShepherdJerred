About

I'm a developer who loves learning new things. Coding is my passion, and I love challenging myself to improve what I can do. I try to open source everything that I can, because I believe that others can help me improve my code, and that my code can help others improve themselves.

Currently I am a sophomore attending Harding University. I am pursuing a BS in computer science, and plan to graduate in 2019. Afterwards I hope to become a software developer for a tech company, or to start a tech company of my own.

I started as a web developer about 4 years ago, mostly using the Twitter Bootstrap CSS framework. I created a couple of paid site templates that had surprising success and remain a great achievement of mine. I recently started using Yahoo's Pure.css CSS library, which I have fallen in love with due to its simplicity.

A little more than a year and a half ago I committed to learning Java. Since then I've created many plugins for the Bukkit API (An extension of the Minecraft server). This has probably been the most important experience in my career, as it taught me many programming concepts, and gave me experience in creating actual applications.

I also enjoy using the Play Framework for backend development, and use it as the backend for a couple of sites, such this website, and a stats page for a Minecraft server I am a developer for.
